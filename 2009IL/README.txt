This directory contains all the files necessary to run ionic liquid
simulations using GROMACS 5.0.7 with the OPLS-2009IL force field.

See papers:
Sambasivarao, S.V.; Acevedo, O. "Development of OPLS-AA Force Field Parameters for 68 Unique Ionic Liquids." 
J. Chem. Theory Comput., 2009, 5, 1038-1050. doi:10.1021/ct900009a

Doherty, B.; Zhong, X.; Gathiaka, S.; Li, B.; Acevedo, O. "Revisiting OPLS Force Field Parameters for Ionic Liquid Simulations."
J. Chem. Theory Comput., 2017, 13, 6131-6145. doi:10.1021/acs.jctc.7b00520
